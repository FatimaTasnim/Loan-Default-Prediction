# Loan Default Prediction
Dataset is collected from [here](https://www.kaggle.com/c/loan-default-prediction).

# Approach:
- Stratified K-Fold crpss validation
- Model: LGBM. (Data tuned until the best accuracy)
# Accuracy:
90%
